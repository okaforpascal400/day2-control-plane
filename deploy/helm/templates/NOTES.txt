Watch it come up:

  kubectl --namespace {{ .Release.Namespace }} get pods -l app.kubernetes.io/instance={{ .Release.Name }} -w

Open the dashboard:

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "day2.fullname" . }}-web {{ .Values.web.service.port }}:{{ .Values.web.service.port }}
  # then browse http://localhost:{{ .Values.web.service.port }}

Talk to the api directly:

  kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "day2.fullname" . }}-api {{ .Values.api.service.port }}:{{ .Values.api.service.port }}
  curl -s http://localhost:{{ .Values.api.service.port }}/health
  curl -s -X POST http://localhost:{{ .Values.api.service.port }}/items \
    -H 'content-type: application/json' -d '{"name":"hello"}'
  curl -s http://localhost:{{ .Values.api.service.port }}/jobs/stats
